Advocate — профессиональный инструмент для адвокатов и юристов.

**Всё под рукой:**
• Клиенты — база с контактами, историей дел и документов
• Дела — полный цикл от открытия до архива
• Документы — сканирование, хранение, быстрый поиск
• Учёт времени — таймер, ручной ввод, отчёты
• Финансы — платежи, расчёты, статистика

**Ключевая фишка — Сканер документов:**
Отсканируй договор, судебный документ или доверенность прямо в приложении. Создай PDF из фото из галереи. Всё документы по делу всегда под рукой — покажи по первому требованию.

**Почему Advocate:**
✓ Работает офлайн — никаких облаков, ваши данные у вас
✓ Быстрый поиск — найди документ за секунды
✓ Учёт времени — точный расчёт с клиентами
✓ PDF экспорт — отправь отчёт или документ одним тапом
✓ Напоминания — не пропусти судебное заседание

**Для кого:**
Адвокаты, юристы частной практики, консультанты, арбитражные управляющие.
