# Advocate iOS — Каркас проекта

## Создано: 2026-05-29

## Что сделано

### 1. Модели данных (SwiftData)
**Файл:** `Models/CoreModels.swift`
- `Client` — клиенты с полным именем, телефоном, email
- `Case` — дела с номером, статусом, методом оплаты
- `Event` — события (встреча, суд, консультация) с типом и длительностью
- `Payment` — платежи по делам
- `Document` — документы с типом, путём к файлу, избранное
- Enums: `CaseStatus`, `PaymentMethod`, `EventType`, `DocumentType` с иконками SF Symbols

### 2. Главный экран (Dashboard)
**Файл:** `Views/ContentView.swift`
- TabView с 5 табами: Главная, Клиенты, Дела, Документы, Время
- Статистика: активные дела, заработано, часы в месяце
- Ближайшие события с иконками типа
- Последние дела со статусом

### 3. Документы (ключевая фича)
**Файл:** `Views/ContentView.swift` (DocumentsView)
- Фильтры по типу документа (горизонтальный скролл)
- Поиск по названию
- Список с иконками типа, датой, избранным
- **Сканер:** VisionKit `VNDocumentCameraViewController`
- **Импорт из галереи:** создание PDF из фото

### 4. App Store метаданные
**Файл:** `AppStore/Metadata.md`
- Название, подзаголовок
- Описание 4000 знаков
- Ключевые слова, категория, цена
- Скриншоты — список что нужно
